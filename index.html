Manual to use

  1. Connect the board to power and connect any switches (transistor, triac, relay) to the board's channels
  2. Take your smartphone and go into list of WI-FI-networks. Connect to "Syeysk_WFR" network
  3. Open a browser you prefer. Go on the address http://192.168.4.1

    You must see this screen. Press these buttons to turn on/off the board's channels

Some tools. And about building the products at home

  • libs_eagle - Libraries of electronic components for EagleCAD
  • webif_libs - JS-libraries for web interfaces
  • tool_webdata2esp - Tool to insert web interface into Arduino program. Written in Python.
  • tool_eagle - Tool to glue PCBs into one big PCB.
